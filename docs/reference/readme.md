# Reference Documents

This folder contains reference documents collected while studying engineering, structural design, and related technologies for Shardhana.

These documents are intended to help understand concepts, terminology, design practices, industrial processes, and engineering fundamentals.

Typical topics include:

- Structural engineering references
- Steel manufacturing processes
- Foundation engineering
- Steel connections
- Industrial plant knowledge
- Design codes and terminology

Reference documents are learning resources.

They do not define Shardhana itself, but provide background knowledge that supports research, development, and practical engineering work.
