Minimal 1D HEM prototype with rule-based localization

We are building a minimal HEM (Han Element Method) prototype in 1D.

Core ideas:

1. Elements are represented as a list of state values.
2. Initial condition is uniform, except for a small perturbation at the center.
3. State evolves over time based on differences with neighboring elements.
4. Differences are amplified (localization), not smoothed.
5. Time stepping is introduced using dt.
6. No FEM equations are used — behavior is rule-based.
7. Necking is represented as increasing concentration of state at a location.

I want to build a very simple 1D prototype of a HEM (Han Element Method) simulation in Python.

Requirements:

1. Represent a 1D bar as a list of elements (for example, 10 elements).
2. Each element has a scalar "state" value with no physical meaning yet.
3. Initialize all elements to 0.0, except the center element, which has a small perturbation such as 0.1.
4. Implement time stepping:

   * Use a time variable t
   * Use a time increment dt
5. At each time step:

   * For each element except the boundaries, compute the average of its two neighbors

   * Define the difference as:

     diff = old_state[i] - neighbor_avg

   * Update the element state using:

     new_state[i] = old_state[i] + dt * k * diff

   * Use a constant k, for example 0.5, as the amplification factor
6. Run the simulation for several time steps.
7. Print the state list at each step so the evolution can be observed clearly.

Important:

* Do NOT use FEM
* Do NOT use stiffness matrices
* This is a rule-based, concept-driven simulation
* Keep the code simple and readable

* Just use plain Python lists and loops
