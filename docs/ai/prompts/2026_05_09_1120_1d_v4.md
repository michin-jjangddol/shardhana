# v4 기본 구조

## Seed

각 Seed는 최소 다음 값을 가짐:

```python
{
    "id": 0,
    "x0": 0.0,
    "x": 0.0,
    "volume0": 1.0,
    "volume": 1.0,
    "fixed": False,
    "role": "normal",
    "state": {
        "damage": 0.0,
        "weakness": 0.0,
    }
}
```

---

## Relation

1D에서는 단순하게:
i 와 i+1 사이를 relation으로 정의

```python
{
    "seed_a": 0,
    "seed_b": 1,
    "d0": 1.0,
    "d": 1.0,
    "k": 1.0,
    "state": {
        "strain": 0.0,
        "damage": 0.0,
        "phase": "elastic",
        "broken": False,
    }
}
```

# 초기 조건

예:

```python
N = 11
spacing = 1.0
volume = 1.0
```

초기 좌표:

```python
x = [0,1,2,3,...]
```

초기 전체 길이:

```python
total_length = N - 1
```

# Boundary Condition

* 첫 Seed:
  fixed = True

* 마지막 Seed:
  role = "control"

현재 단계에서는:
오른쪽 끝 seed만 이동

예:

```python
x += displacement_rate
```
