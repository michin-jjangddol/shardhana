Shardhana Project

An open-source project for structural analysis inspired by nature

Project Overview

The Shardhana Project goes beyond traditional FEM (Finite Element Method) by applying AEM (Applied Element Method) for a more intuitive and seamless structural analysis. We aim to build tools that help understand and analyze the behavior of structures in nature, together with developers around the world.

Goals & Vision

  • Develop an open-source structural analysis engine based on AEM
  • Build a collaborative GitHub-based global community
  • Grow a sustainable open community
  • Provide free tools and education for future engineers

Shardhana Declaration

"We strive to understand human-made structures as part of nature, and through open collaboration, create a new path in structural analysis."

Progress
